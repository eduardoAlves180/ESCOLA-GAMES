# ESCOLA-GAMES
 Um jogo desenvolvido com **HTML, CSS e JavaScript** que une diversão e aprendizado. Criado para estimular o raciocínio, a memória e outras habilidades das crianças por meio de atividades interativas e desafios educativos em uma interface simples, colorida e envolvente.
